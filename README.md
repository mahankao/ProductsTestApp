# My Next.js Products App

This is a **Next.js** project for managing a list of products.  
It includes full **CRUD functionality**, search, favorites, and pagination. Built with TypeScript and Zustand 
---

## Features

- View all products
- Mark products as **favorites**
- Search products by title
- Add new products
- Edit existing products
- Delete products
- Pagination
- Responsive design for mobile and desktop
- State management with Zustand

---

## Stack

- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Zustand](https://zustand-demo.pmnd.rs/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide Icons](https://lucide.dev/)
- Fetching products from [FakeStore API](https://fakestoreapi.com/)

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/mahankao/ProductsTestApp.git
```
Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result
